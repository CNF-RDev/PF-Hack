# CNF's PF-Hack
A free and open-source Gui for Phantom Forces
------------------


Features:
  * Aimbot
  * ESP
  * Silent Aim
  * Colors (although idk if that really counts as a feature)

-----

F - Toggle Gui

Right Mouse - Toggle Aimbot
